Hızlı Başlangıç

Birkaç komutla ayağa kaldırın

macOS üzerinde hazır (prebuilt) Docker imajı ile Mercury'yi dakikalar içinde çalıştırın.

Kurulum Akışı

Web arayüzü özel (proprietary) olduğundan kaynaktan derlenmez; önerilen yöntem GHCR'deki hazır imajı çekmektir. Depoyu yalnızca compose dosyası, scriptler ve iOS sağlayıcı için klonlarsınız.

  • 1 Gereksinimleri kurun (Docker, Node, Xcode araçları)
  • 2 Depoyu klonlayın ve host araçlarını kurun (npm ci)
  • 3 Hazır imajı çekip stack'i başlatın, iOS sağlayıcıyı ayağa kaldırın
  • 4 Tarayıcıdan https://<MERCURY_DOMAIN> adresine gidin
# 1) Depoyu klonla (compose, script, iOS sağlayıcı) git clone https://github.com/erdncyz/mercury-farm.git cd mercury-farm # 2) Host araçlarını kur npm ci # 3) Hazır imajı çek ve stack'i başlat npm run stack:up:image:macos ./scripts/start-ios-provider.sh # 4) Arayüzü aç → https://<MERCURY_DOMAIN>
